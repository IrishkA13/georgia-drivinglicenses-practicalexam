Script to get appointment for driving practicalexam in Georgia for available dates.
Fill in your personal data to register for the driving test in the input.properties file.
Script checks every 5 sec for available exam time for input dates.
The reservation is waiting for your payment for 1 hour. You need to check computer
screen every 1 hour.

Скрипт для записи на практический экзамен по вождению в Грузии на доступные даты.
Заполните свои личные данные для записи на экзамен по вождению в файле input.properties.
Скрипт проверяет наличие свободного времени для экзамена каждые 5 секунд.
Бронирование ожидает оплаты в течение 1 часа. Вам необходимо проверять экран компьютера каждый 1 час.

Всем привет!
Я программист, и когда сдавала экзамен в городе, написала скрипт, чтобы отлавливать запись на экзамен в подходящие мне даты. Хочу поделиться этой возможностью со всеми.
Скрипт нужно запускать на своем компьютере, он постоянно проверяет наличие времени на экзамен в промежутке указанных дат и при наличии времени кликает оплатить - у вас есть 1 час на оплату, то есть удобно запускать скрипт, когда вы сидите за компьютером или готовы подходить к компьютеру раз в час.

Для непрограммистов...
1) у кого Mac - я сделала обертку GeorgiaDrivingExam.zip
    - личные данные для записи на экзамен необходимо указать в файле input.properties (открыть файл, ввести нужные вам данные, сохранить файл);
    - правой клавишей кликнуть GeorgiaDrivingExam, New Terminal at folder - далее введите "cd .." и Enter, далее введите "xattr -dr com.apple.quarantine GeorgiaDrivingExam.app" и Enter (Apple блокирует запуск неизвестного бинарника, эта команда позволяет снять карантин с файла)
    - дважды кликнуть run.sh - откроется окно chrome-браузера и скрипт начнет работу;
2) у кого Windows - у меня нету Windows, поэтому не могу сделать обертку; нахожусь в Батуми; если готовы подойти со своим Windows - можем сделать обертку вместе;
3) у кого Linux - круто, уверена, вы разберетесь со ссылкой для программистов;

Файл input.properties содержит данные, которые вы вводите для записи на экзамен в браузере
personalNumber - ваш паспорт или другой документ
phoneNumber - номер телефона
category - автомат/механика грузинскими буквами как на сайте
city - город грузинскими буквами, где планируете сдавать экзамен
minDateToSelect - минимальная дата, начиная с которой вы готовы прийти на экзамен
maxDateToSelect - крайняя дата, заканчивая которой вы готовы прийти на экзамен
Когда есть доступное время в указанные даты, бронируется первое доступное время.
